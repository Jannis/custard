# CUSTARD

[MUSTARD](http://github.com/CodethinkLabs/mustard) implemented in Clojure.

## Get started locally

### Create a CUSTARD repository

```
mkdir ~/your-project
cd ~/your-project
git init
```

### Clone CUSTARD

```
git clone https://github.com/jannis/custard
```

### Start CUSTARD

```
cd custard
CUSTARD_PATH=~/your-project boot run-development
```
